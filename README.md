# AI-powered Resume Screening and Candidate Ranking System

A machine learning system that automatically screens resumes and ranks candidates based on their suitability for a given job posting. Built as a machine learning project using Python and Streamlit.

---

## Key Features

- Upload a resume in PDF or TXT format
- Select a job role from thousands of real job postings
- Predicts candidate suitability: **Highly Suitable**, **Suitable**, or **Not Suitable**
- Displays model confidence score and cosine similarity between resume and job
- Export evaluation results as a CSV report

---

## Technologies Used

| Tool | Purpose |
|------|---------|
| Python 3 | Core programming language |
| Streamlit | Web application interface |
| scikit-learn | Machine learning models and TF-IDF |
| pandas / numpy | Data processing |

| matplotlib | Evaluation charts |
| joblib | Model serialization |

---

## Machine Learning Models Used

| Model | Role |
|-------|------|
| Logistic Regression | Baseline classifier |
| Decision Tree | Baseline classifier |
| Random Forest | Final model (best performance) |
| GridSearchCV | Hyperparameter tuning for Random Forest |
| TF-IDF Vectorizer | Feature extraction (5,000 features) |
| Cosine Similarity | Resume-job similarity scoring and label generation |

The best-performing model (Random Forest with GridSearchCV tuning) is saved and used in the web application.

---

## Dataset Information

| Dataset | Description |
|---------|-------------|
| `UpdatedResumeDataSet.csv` | Resume text with job category labels |
| `job_postings.csv` | Real job postings (title, description, skills, experience level) |

Training pairs are generated by combining each resume with a sampled set of job postings and labeling them using cosine similarity thresholds:
- **Highly Suitable**: similarity ≥ 0.10
- **Suitable**: similarity ≥ 0.05
- **Not Suitable**: similarity < 0.05

## Installation Steps

**1. Clone the repository:**
```bash
git clone https://github.com/suniltech1/AI-Resume-Screening-Candidate-Ranking.git
cd AI-Resume-Screening-Candidate-Ranking
```

**2. Create and activate a virtual environment:**
```bash
python -m venv .venv
.venv\Scripts\activate        # Windows
source .venv/bin/activate     # macOS/Linux
```

**3. Install dependencies:**
```bash
pip install -r requirements.txt
```

---

## How to Run the Project

Run the following scripts in order:

**Step 1 — Preprocess data:**
```bash
python preprocessing.py
```

**Step 2 — Train models:**
```bash
python train_model.py
```

**Step 3 — Evaluate the model:**
```bash
python evaluate.py
```

**Step 4 — Launch the web application:**
```bash
streamlit run app.py
```

Then open your browser at `http://localhost:8501`.
